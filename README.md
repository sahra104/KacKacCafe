# Kac Kac Café

## About the Project

Kac Kac Café is a modern Somali-themed pastry and café website built using React, TypeScript, and Vite.

The website celebrates Somali culture through traditional pastries, tea culture, hospitality, and authentic cuisine.

---

## Features

- Responsive design
- Somali cultural theme
- Modern café aesthetic
- Multiple pages
- React Router navigation
- Contact form
- Gallery section
- Services page
- Blog page
- Menu showcase

---

## Technologies Used

- React
- TypeScript
- Vite
- CSS3
- HTML5

---

## Pages

### Home
Introduction to Kac Kac Café.

### About
History and story of the café.

### Menu
Traditional Somali pastries and drinks.

### Gallery
Photos of food and café interiors.

### Services
Catering and event services.

### Blog
Articles about Somali food culture.

### Contact
Customer contact form.

---

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
