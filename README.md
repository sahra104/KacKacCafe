# KacKacCafe

An authentic web representation of Kac Kac Café based in Nairobi, Kenya. This platform features our cultural heritage, traditional pastries (Kac Kac, Malawax, Sambuus), and custom spiced hot beverages.

## Features
- Beautiful layout pages for Home, About, Menu, Services, Gallery, and Blog.
- Fully working interactive user contact form.
- Structured code using modern React components.
