# KacKacCafe 🍵

An authentic web representation of **Kac Kac Café** based in Nairobi, Kenya. This platform showcases our Somali cultural heritage, traditional pastries, and custom spiced hot beverages.

## Pages
- **Home** – Hero carousel with cultural imagery and welcome section
- **About** – Our story, heritage, and preparation traditions
- **Menu** – Full menu with live search/filter feature
- **Services** – Cultural gatherings, corporate catering, custom pickups
- **Gallery** – Filterable photo gallery (Pastries, Beverages, Culture)
- **Blog** – Culinary journal articles on spice techniques and baking
- **Contact** – Validated contact and booking form

## Features
- Fully responsive layout using Bootstrap 5.3
- Sticky navbar with active page highlighting
- Hero image carousel (index page)
- **JavaScript Feature 1:** Live menu search — filters items in real time as you type
- **JavaScript Feature 2:** Gallery category filter — show all, pastries, beverages, or culture
- **JavaScript Feature 3:** Contact form validation — checks all fields before submission
- Consistent warm Somali-inspired colour theme across all pages
- Real cultural images in `assets/` folder

## Tech Stack
- HTML5, CSS3, Bootstrap 5.3
- Vanilla JavaScript
- Google Fonts (Poppins)
